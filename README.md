# [Sing-Box DNS Configurator](https://kira361323.github.io/sing-box-generator/)

Инструмент для генерации конфигурационных файлов Sing-Box с поддержкой кастомной маршрутизации и управления логированием.

## Функционал

* **Гибкая маршрутизация**: Разделение трафика между Redirect DNS и Direct DNS.
* **Кастомизация**: Поддержка добавления собственных доменов и приложений (пакетов) для роутинга.
* **Исключения**: Возможность полного обхода туннеля для конкретных приложений и доменов.
* **Управление логированием**: Настройка `log.level` (от `panic` до `trace`) для отладки и экономии ресурсов.
* **Профили**: 
    * `Bench` — для теста скорости DNS-серверов.
    * `Dynamic` — автоматический выбор самого быстрого сервера (Load Balancing).
    * `Fixed` — ручной выбор конкретных серверов.

## Использование

1. Открой `index.html` в любом современном браузере.
2. Выбери нужный уровень логирования.
3. При необходимости укажи домены и пакеты приложений в соответствующих полях.
4. Выбери режим генерации (Bench, Dynamic или Fixed).
5. Нажми кнопку для автоматического скачивания сформированного `.json` файла.

## Технические детали

* **Интерфейс**: Чистый HTML/CSS/JS (Vanilla), без внешних зависимостей.
* **Формат данных**: Стандартный JSON, совместимый с актуальными версиями Sing-Box.
* **Маршрутизация**: Использование удаленных списков `domains.srs` для эффективной работы с DNS.
