Sing-Box DNS Configurator

Системные параметры

Remote Rule-Set используется как основной источник.

При включённом cache_file удалённый Rule-Set кешируется sing-box.
При последующих запусках сохранённый кеш позволяет использовать ранее загруженный Rule-Set.
Локальный domains.srs в конфигурацию не добавляется.
Не добавляйте путь к локальному domains.srs: если файла нет в каталоге приложения, sing-box завершит инициализацию с ошибкой "no such file or directory".
