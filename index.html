Sing-Box DNS Configurator

Системные параметры

Remote Rule-Set используется как источник списка доменов.

При первом запуске sing-box скачивает domains.srs и сохраняет его в cache.db.
При следующих запусках sing-box использует сохранённый кеш, пока не будет получена новая версия.
Если обновление Remote Rule-Set временно недоступно, предыдущая закешированная версия используется автоматически.
Локальный domains.srs напрямую в конфигурацию не добавляется, поэтому ошибка отсутствующего локального файла не возникает.
Не указывайте domains.srs как type: local, если файл физически не существует в каталоге приложения.

Выбор DNS-серверов

Вся система по умолчанию использует Direct DNS.
Приложения и домены, указанные ниже, используют Redirect DNS.

Маршрутизация в Redirect DNS

Remote Rule-Set → Redirect DNS

Все домены, содержащиеся в удалённом domains.srs, автоматически используют Redirect DNS.

Порядок обработки:
1. Android-приложения из списка → Redirect DNS
2. Домены из списка → Redirect DNS
3. Домены из domains.srs → Redirect DNS
4. Все остальные DNS-запросы → Direct DNS

Исключения

Эти домены имеют приоритет над Remote Rule-Set и используют Direct DNS.
Трафик этих Android-приложений исключается из TUN и отправляется напрямую.
Готово к генерации.
