Sing-Box DNS Configurator

v3.7 sing-box 1.11 – 1.14+ Android · Windows · Linux offline
01

Платформа, режим и версия синтаксиса

Android — маршрутизация по package_name, исключение приложений из TUN через exclude_package.
Windows / Linux — маршрутизация по process_name. Исключения приложений и чекбокс WhatsApp доступны только на Android.
Фиксированный — один Redirect и один Direct DNS.
Fallback — все DNS категории в конфиге, перебор при таймауте. Порядок = приоритет.
Modern — rule-set через http_client: {"detour":"direct"}, без warning на 1.14+.
Legacy — rule-set через download_detour для устройств на 1.11–1.13.
02

Системные параметры

Отключены — "log":{"disabled":true}. panic — продакшен. debug/trace — только отладка.
Частота загрузки .srs-файла с доменами.
UDP-DNS (:53), резолвит hostname DoH/DoT-серверов. Проверяй через dig @IP xbox-dns.ru.
IPv4 only убирает таймауты, если провайдер не маршрутизирует IPv6.
Если на WiFi приложения резолвятся, но данные не проходят — снизь до 1280/1360 (проверка MTU-гипотезы).
Пусто = ссылка по умолчанию (Dns-rules). Иначе — свой .srs.
