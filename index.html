Sing-Box DNS Configurator v3

Системные параметры

Используется для резолва доменных имён DoH/DoT-серверов. Выберите IP, доступный в вашей сети. При недоступности bootstrap — DNS-профиль не работает.
Приложения исключаются напрямую из TUN-интерфейса (bypassing VPN). Удаленный Rule-Set загружается с параметром download_detour: "direct".

Режим работы

Фиксированный: вы выбираете конкретный Redirect и Direct DNS.
Fallback: все DNS из списка добавляются в конфиг. sing-box использует первый доступный, при таймауте переключается на следующий. Порядок = приоритет.

Выбор DNS-серверов

Обслуживает выбранные приложения, домены и Remote Rule-Set.
Обслуживает весь остальной трафик.

Маршрутизация в Redirect DNS

Исключения

Исключённые домены маршрутизируются через выбранный Direct DNS (зашифрованный DoH/DoT), а не через DNS оператора.
Указанные приложения добавляются в inbounds.exclude_package и полностью минуют VPN-туннель.
Готово к генерации. Выберите режим и DNS-серверы.
