Sing-Box DNS Configurator

Системные параметры

Режим Remote + Cache:
1. При первом запуске sing-box загружает Remote Rule-Set.
2. Загруженный Rule-Set сохраняется в cache.
3. При последующих запусках sing-box проверяет обновление согласно выбранному интервалу.
4. Если обновление недоступно, используется сохранённая версия из cache.
5. Локальный domains.srs включён в конфигурацию как отдельный резервный Rule-Set.
Важно: sing-box не предоставляет стандартного JSON-механизма автоматического переключения Remote → Cache → Local. Cache для Remote Rule-Set обрабатывается самим sing-box. Local Rule-Set является отдельным источником.
